# Particle Conquest Pro

A real-time strategy particle game where you compete to conquer territory by growing your particle swarm.

## Play

Open `index.html` in any modern browser to play - no installation required.

## Features

- Touch and mouse controls
- Special abilities with cooldowns
- AI opponent
- Smooth particle physics
- Mobile-friendly design

## Controls

- Click/tap to direct your particles
- Use ability buttons to activate special powers
